# Codex for Legal Codex Templates

Each practice-area folder contains an upstream `CLAUDE.md` playbook template rewritten for Codex-local paths.

Copy the template you need into `config/local/codex-for-legal/<practice>/CLAUDE.md` and customize it with your house style, authority matrix, risk preferences, and matter defaults. `config/local/` is gitignored because those files can contain confidential legal preferences and matter details.
